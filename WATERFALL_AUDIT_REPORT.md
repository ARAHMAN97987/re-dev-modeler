## الجزء 3: الهيكل لكل مرحلة (Per-Phase)

### النتيجة: ✅ مُتحقق

الهندسة المعمارية:
```
لكل مرحلة:
  1. CAPEX المرحلة = مجموع أصول المرحلة
  2. تكلفة التطوير = CAPEX + حصة الأرض المخصصة
  3. حقوق ملكية = تكلفة التطوير × (1 - LTV%)
  4. دين = تكلفة التطوير × LTV%
  5. رسوم = بناءً على حقوق ملكية/CAPEX المرحلة
  6. Waterfall = مستقل لكل مرحلة
  7. LP/GP IRR = IRR لصافي التدفقات لكل مرحلة
```

التجميع:
```
Consolidated LP IRR = IRR(مجموع تدفقات LP لكل المراحل)
وليس: متوسط IRRs المراحل
```

### Tier 3 (GP Catch-up): ✅ سليم

- الافتراضي `"perYear"` يطابق Excel ✅

### Tier 4 (Profit Split): ✅ سليم
